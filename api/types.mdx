## Interfaces

### GameState

Represents the complete game state for a room.

```typescript
interface GameState {
  users: UsersEntity[];
  trumpCard: number;
  state: string;
  leadPlayer: number;
  turn: number;
  scores: number[];
}
```

<ResponseField name="users" type="UsersEntity[]" required>
  Array of all players in the game with their complete state
</ResponseField>

<ResponseField name="trumpCard" type="number" required>
  The trump card for the current round (1-49, or 0 if not set)
</ResponseField>

<ResponseField name="state" type="string" required>
  Current game phase: "lobby", "cardSwap", "inGame", or "end"
</ResponseField>

<ResponseField name="leadPlayer" type="number" required>
  Index of the player leading the current trick (-1 if not set)
</ResponseField>

<ResponseField name="turn" type="number" required>
  Current turn offset from lead player (0-3, or 4 when trick complete)
</ResponseField>

<ResponseField name="scores" type="number[]" required>
  Array of two team scores [team1Score, team2Score]
</ResponseField>

**Example:**
```typescript
const gameState: GameState = {
  users: [
    // UsersEntity objects
  ],
  trumpCard: 25,
  state: "inGame",
  leadPlayer: 0,
  turn: 2,
  scores: [3, 1]
}
```

### UsersEntity

Represents a player's complete state within a game.

```typescript
interface UsersEntity {
  id: string;
  name: string;
  toSwap: number[];
  cardPlayed: number;
  hand: number[];
  team: number;
  tricksWon: number[][];
}
```

<ResponseField name="id" type="string" required>
  Unique identifier for the user
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the player
</ResponseField>

<ResponseField name="toSwap" type="number[]" required>
  Array of 3 card numbers selected for swapping (empty if not yet selected)
</ResponseField>

<ResponseField name="cardPlayed" type="number" required>
  Card number currently played in the trick (0 if no card played)
</ResponseField>

<ResponseField name="hand" type="number[]" required>
  Array of card numbers in player's hand (sorted ascending)
</ResponseField>

<ResponseField name="team" type="number" required>
  Team assignment (0 = no team, 1 = team 1, 2 = team 2)
</ResponseField>

<ResponseField name="tricksWon" type="number[][]" required>
  Array of tricks won, where each trick is an array of 4 card numbers
</ResponseField>

**Example:**
```typescript
const player: UsersEntity = {
  id: "user123",
  name: "Alice",
  toSwap: [5, 12, 23],
  cardPlayed: 0,
  hand: [1, 7, 14, 21, 28, 35, 42, 49],
  team: 1,
  tricksWon: [
    [1, 8, 15, 22],
    [7, 14, 21, 28]
  ]
}
```

## Card Representation

Cards are represented as numbers from 1-49:

<Expandable title="Card Number System">
  **Card Numbering:**
  - 0: Empty/no card
  - 1-7: Green suit (1=Ace, 7=Seven)
  - 8-14: Purple suit
  - 15-21: Pink suit
  - 22-28: Yellow suit
  - 29-35: Black suit
  - 36-42: Red suit
  - 43-49: Blue suit

  **Special Cards:**
  - Card 1: Green Ace (always wins tricks)
  - Card 49: Blue 7 (highest card, leads first trick if no Ace)
  - Cards ending in 7 (7, 14, 21, 28, 35, 42, 49): Sevens (scoring cards)

  **Formulas:**
  ```javascript
  // Get suit (1-7)
  suit = Math.floor((card - 1) / 7) + 1

  // Get rank (1-7 within suit)
  rank = card - 6 * (suit - 1)
  ```
</Expandable>

## State Flow

<Expandable title="Game State Transitions">
  **State Progression:**
  
  1. **lobby**: Initial state when room is created
     - Players join and select teams
     - Transitions to "cardSwap" when `startGame()` called
  
  2. **cardSwap**: Card swapping phase
     - Each player selects 3 cards to swap with partner
     - Transitions to "inGame" when all swaps complete
  
  3. **inGame**: Active gameplay
     - Players play cards in tricks
     - Transitions back to "cardSwap" when round ends
  
  4. **end**: Game completed
     - One team has won overall
</Expandable>
